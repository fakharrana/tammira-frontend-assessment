This project implements the required features for the assessment, please refer to the [document](https://docs.google.com/document/d/147zgWv3uuGxmFhDRm5dkpriFxvjPo1gEEekOZihpjVw/edit?tab=t.0#heading=h.akvmfnuoi19d). 

---

## Installation  

### 1. Clone the Repository  
```bash
git clone https://github.com/fakharrana/tammira-frontend-assessment.git
```

### 2. Install Dependencies 
```bash
npm install
```

### 3. Set up Environment Variables (.env)
```bash
API_BASE_URL=http://localhost:3000/api
```
---

## Usage 

### 1. Start Metro Bundler
```bash
npm start
```

### 2. Run Application
```bash
npm run ios

OR

npm run android
```
